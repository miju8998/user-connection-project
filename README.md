# User Connection Project

<h2>Description</h2>
<p>This project demonstrates PHP abstract classes, interfaces, traits, namespaces, password hashing, sessions, and MySQL database storage.</p>

<h2>Features</h2>
<ul>
  <li>User registration</li>
  <li>User login and logout</li>
  <li>Admin and regular user roles</li>
  <li>Password hashing</li>
  <li>Data stored in MySQL database</li>
</ul>
